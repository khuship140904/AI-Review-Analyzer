🤖 AI Review Analyzer

An AI-based project that analyzes user reviews and determines their sentiment (Positive, Negative, or Neutral). This helps businesses understand customer feedback efficiently.

---

🚀 Features

* 🔍 Sentiment analysis of reviews
* 📊 Classifies reviews as Positive / Negative / Neutral
* ⚡ Fast and automated processing
* 🧠 Uses Machine Learning / AI techniques

---


📸 Output Example

* Input: "This product is amazing!"
* Output: Positive ✅
